# 无人机实时监控系统 - 系统架构文档

## 1. 系统概述

本系统是一个基于Web的无人机实时监控系统，采用前后端分离架构，集成YOLOv8目标检测算法，实现用户认证、文件上传、实时检测、结果展示等功能。

## 3. 分层架构详解

### 3.1 表现层 (Presentation Layer)

#### 3.1.1 前端页面
- **login.html - 登录页面
- **register.html** - 注册页面
- **monitor.html** - 主监控页面

#### 3.1.2 前端资源
- **CSS样式** (style.css) - 页面样式定义
- **JavaScript** (main.js) - 页面交互逻辑

#### 3.1.3 主要功能
- 文件上传预览
- 检测结果展示
- 历史记录管理
- 模型切换交互
- 视频流显示

### 3.2 应用层 (Application Layer)

#### 3.2.1 Flask应用 (app.py)
**主要路由配置
- 用户认证路由
- 监控页面路由
- 文件上传路由
- 检测API路由
- 视频流路由

**核心组件
- VideoStreamManager - 视频流管理器
- 会话管理
- 请求处理

### 3.3 业务逻辑层 (Business Logic Layer)

#### 3.3.1 ModelManager类 (model_manager.py)
**职责**
- 模型加载与管理
- 图片/视频检测
- 结果处理

**核心方法**
```python
class ModelManager:
    - __init__() - 初始化，加载模型
    - predict_image() - 图片预测
    - process_video_stream() - 视频流处理
    - crop_detection_from_frame() - 裁剪检测目标
    - draw_detections() - 绘制检测框
```

**关键特性**
- 支持多模型切换
- 跳帧检测优化
- 检测结果缓存

#### 3.3.2 SPD模块注册 (register_spd.py)
**功能**
- 动态注册SPD卷积模块
- 解决自定义模块加载问题

### 3.4 数据访问层 (Data Access Layer)

#### 3.4.1 数据库操作 (database.py)
**数据库表结构**
```
users表:
- id (主键
- username (用户名)
- password_hash (密码哈希)
- created_at (创建时间)
```

**主要操作
- 用户创建
- 用户验证
- 管理员检查

#### 3.4.2 文件系统
- static/uploads/ - 上传文件存储
- static/crops/ - 检测目标裁剪图存储

### 3.5 基础设施层 (Infrastructure Layer)

#### 3.5.1 外部依赖
- YOLOv8 (Ultralytics) - 目标检测引擎
- OpenCV - 视频图像处理
- PIL - 图像预处理
- SQLite - 数据库

## 4. 数据流设计

### 4.1 图片检测数据流
```
用户选择图片 
    ↓
文件上传到服务器
    ↓
保存到uploads目录
    ↓
调用ModelManager.predict_image()
    ↓
YOLOv8推理
    ↓
返回检测结果
    ↓
绘制检测框
    ↓
裁剪目标图片
    ↓
返回检测结果JSON
    ↓
前端展示结果
```

### 4.2 视频检测数据流
```
用户选择视频
    ↓
文件上传到服务器
    ↓
保存到uploads目录
    ↓
调用/api/detect - 获取第一帧检测结果
    ↓
启动VideoStreamManager
    ↓
后台线程读取视频帧
    ↓
ModelManager.process_video_stream()
    ↓
跳帧检测 + 帧插值
    ↓
编码为MJPEG流
    ↓
前端展示视频流
    ↓
轮询获取检测结果
    ↓
更新检测列表
```

## 5. 关键技术实现

### 5.1 跳帧检测算法
```
视频流
    ↓
记录当前时间
    ↓
判断是否满足检测间隔?
    ├─ 是 → 执行YOLOv8推理
    │         ↓
    │      更新检测结果
    │         ↓
    └─ 否 → 复用上次结果
              ↓
         绘制检测框
              ↓
         返回处理帧
```
